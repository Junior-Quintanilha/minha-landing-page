Projetos

Sistema Bancário em Java

Desenvolvimento de uma aplicação de console em Java que simula as operações essenciais de um sistema bancário. O projeto aplica conceitos fundamentais de Orientação a Objetos (Encapsulamento) para realizar depósitos, saques e transferências, garantindo a validação de saldos.

Sistema de Biblioteca em C

Projeto da faculdade de gerenciamento de biblioteca desenvolvido na linguagem C, utilizando structs. A aplicação permite cadastro, exibir livros cadastrados, pesquisa por código e ordenar livro por ano de publicação.

Landing Page com HTML e CSS

Projeto da facultade de desenvolvimento de uma Landing Page responsiva utilizando apenas HTML5 e CSS3 sem a necessidade de frameworks. O projeto foca na estruturação semântica do código, aplicação de layouts modernos com Flexbox e CSS Grid, e implementação de Media Queries para garantir uma experiência de usuário (UX) otimizada em dispositivos móveis e desktops.
